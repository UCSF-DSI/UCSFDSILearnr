What is Descriptive Statistics?

R is often referred to as a “statistical” programming language because it is often used for statistics. Having an understanding of statistical thinking is essential to doing data science work with R.

We define statistics is using math to make sense of data. There are two general types of statistics:

  • Descriptive Statistics: Using statistics to describe your data.

  • Inferential Statistics: Using statistics to infer information beyond your data.

In this module, we’ll focus on descriptive statistics, so if the definition of inferential statistics doesn’t make complete sense to you yet, that’s okay! In R for Statistics - Part 2: Inferential Statistics, we’ll spend more time differentiating between descriptive and inferential statistics.

The first thing you’ll learn to describe in this data are its variables. Statistical variables are characteristics or attributes of an entity of interest, and they are usually represented as columns within a data table. It’s important to note that this is different from the object/variable concept in computer programming, and when the term variable is used, you should pay attention to whether it’s being used in a statistical or computer programming context.

In this data, the entity of interest (represented as each row) is billionaires, and the table below lists the variables associated with billionaires.

Column Description (Taken from CORGIS)
name “The name of the billionaire.”
company_founded “The year that the company was founded.”
company_sector “The sector of the business, or what segment of the economy they fit into.”
age “The current age of the billionaire. Ages that are represented as -1 stand for ages that were not available in the data that was collected.”
gender “A string representing their gender.”
citizenship_country “The name of the country that this billionaire has citizenship with.”
net_worth “The number of billion of dollars that this billionaire is worth.”
inheritance_source “The way that this money was inherited (or not inherited). Inheritance can come from a spouse, the father, or from multiple generations within a family (either 3, 4, or 5+).”

There are two main types of statistical variables: numerical and categorical.

Numerical variables are variables that contains numbers. There are two subtypes of numerical variables: continuous and discrete. Continuous variables are variables where there can be an infinite number of values between any two values. Usually, variables that contain decimal numbers are continuous variables. Discrete variables are variables where values are separated by specific intervals. Any integer-based variables is an example of discrete variables.

Categorical Variables are variables that do not contain numbers. The two types of categorical variables are ordinal and nominal. Ordinal variables contain categories that are ordered. A common example of an ordinal variable are Likert scales and other types of ratings. Nominal variables contain categories that are not ordered.

Below is a summary of the types of statistical variables:

The main types of statistical variables are: numerical - discrete, numerical - continuous, categorical - ordinal, and categorical - nominal

It’s important to identify the type of variables you are working with because it provides a framework to determine the type of analysis you can apply to the variable. In the next few lessons, you’ll learn about different ways of applying descriptive statistics, and the types of analysis you can apply to a numerical variable are different than your options for a categorical variable.

Other Variable Types

In addition to the types of variables above, there are a few special subtypes of numeric and categorical variables you should be aware of:

  • Time series is a type of continuous (numeric) variable based on time.

  • Raw text is a type of categorical variable based on free-form text entry. Natural Language Processing (NLP) is an entire field focused on the processing and analysis of raw text and natural language. Although this module will not discuss NLP, you should be aware that there are analysis techniques available to handle this type of variable.

  • Image data consists of discrete (numeric) variables representing the color of each pixel. Computer Vision is an entire field focused on analyzing and interpreting image data. Similar to NLP, Computer Vision will not be discussed in this module, but you should be aware that analysis techniques exist for image data as well.

Visualizations

Visualizations can be used to describe the values that can appear in numeric variables. For continuous variables, we can use histograms, and for discrete variables, we can use bar charts.

Histograms

Histograms display continuous variables by representing the frequency of ranges of values as bars. Run the code below to display a histogram of net_worth variable.

# Create a histogram of the net_worth variable with 30 bins and assign it to net_worth_hist
net_worth_hist <- ggplot(billionaires) + 
  aes(x= net_worth) + 
  geom_histogram(bins = 30) +
  ggtitle("Histogram of Billionaire Net Worths") +
  xlab("Net Worth (Billions)") + 
  ylab("Count")

# Display net_worth_hist
net_worth_hist

From the histogram, we can see that most of the net worth values are below 10 billion. Let’s break down how this code works.

We are using the ggplot2 library, which is based on the grammar of graphics. The grammar of graphics is a way to describe visualizations. We won’t dive too deep into the grammar of graphics in this module, but you will learn more about it in the R for Visualizations series (coming soon)!

The first step to create your histogram is to initiate a new ggplot using the ggplot function. In the code sample below, we are initiating a ggplot for the data in our billionaires dataframe and assigning it to an object called net_worth_hist.

# Initiate a new ggplot based on billionaires and assign to net_worth_hist
net_worth_hist <- ggplot(billionaires)

# Display net_worth_hist
net_worth_hist

Right now, net_worth_hist is just an empty plot because we haven’t added anything to it yet. The first thing we’ll add to net_worth_hist is an aesthetic mapping. An aesthetic mapping maps variables from your data to visual properties of your plot. In this case, we want our x-axis to represent the net_worth variable from billionaires, and we’ll use the aes function to accomplish that.

# Initiate a new ggplot based on billionaires and assign to net_worth_hist
net_worth_hist <- ggplot(billionaires) +
  aes(x = net_worth) # map net_worth to the x axis

# Display net_worth_hist
net_worth_hist

Now we can see net_worth in the x-axis, but we still need to plot our data. We’ll do this by adding a geometry. Think of geometries as layers of drawings. The first geometry we’ll add is geom_histogram which will create a histogram.

# Initiate a new ggplot based on billionaires and assign to net_worth_hist
net_worth_hist <- ggplot(billionaires) +
  aes(x = net_worth) + # map net_worth to the x axis
  geom_histogram(bins = 30) # create a histogram with 30 bins

# Display net_worth_hist
net_worth_hist

From this point, we can add a title and adjust axes labels using ggtitle, xlab, and ylab.

# Create a histogram of the net_worth variable with 30 bins and assign it to net_worth_hist

# Display net_worth_hist

In summary, to create ggplots, you’ll need to first initialize it, and then you’ll add layers to your plot based on what you need to visualize.

Bar Chart

Bar Charts are similar to histograms, but instead of displaying frequencies of ranges of values, bar charts display frequencies for specific values. You should use Bar Charts when working with discrete variables.

Similar to histograms, bar charts can also be created with ggplot. The main difference is you’ll use geom_bar() instead of geom_histogram().

# Filter the billionaires to only include data that provided age (hint: what does age == -1 mean?). Assign the results to billionaires_with_age
billionaires_with_age <- billionaires %>%
  filter(age > 0)

# Create a bar plot of the age variable with a bar width of 0.8 and assign it to age_bar
age_bar <- ggplot(billionaires_with_age) + 
  aes(x = age) +
  geom_bar(width = 0.8) + 
  ggtitle("Bar Plot of Billionaire Ages") +

Measures of Center

As the name implies, measures of center are ways to describe the “center” value of a variable. They are important because they also serve as a proxy to represent the expected or typical value of a variable. The three measures of center you’ll learn about in this lesson are:

  • Mean: Also known as average, the mean is the sum of all values divided by the number of values. You can calculate mean using the mean() function. Mean can be used with both continuous and discrete data.

  • Median: The median is the value that separates the top and bottom halves of the data. You can calculate median using the median() function. Median can be used with both continuous and discrete data.

Let’s see what these look like in our histogram of company_founded (company_founded_bar). Let’s start with the mean.

# Filter the billionaires to only include data company_founded > 1500
billionaires_more_recent <- billionaires %>%
  filter(company_founded > 1500)

# Create a bar plot of the company_founded variable and assign it to company_founded_bar
company_founded_bar <- ggplot(billionaires_more_recent) + 
  aes(x = company_founded) +
  geom_bar(width = 0.8) + 
  ggtitle("Bar Plot of company_founded") +
  xlab("Year Company Founded") + 
  ylab("Count")

# Display mean of company_founded
company_founded_bar <- company_founded_bar + 
  geom_vline(xintercept = mean(billionaires_more_recent$company_founded), color = "red")

# Display company_founded_bar
company_founded_bar

# Print mean of net_worth
print(sprintf("The mean is %s", mean(billionaires_more_recent$company_founded)))

Here we used geom_vline() to create a red vertical line at the mean. Now let’s create a blue line to represent the median.

# Filter the billionaires to only include data company_founded > 1500

Vocabulary Definition
Statistics Using math to make sense of data.
Descriptive Statistics Use statistics to describe your data.
Inferential Statistics Use statistics to infer information beyond your data.
Variables (Statistics) Characteristics or attributes of an entity of interest. These are usually represented as columns within a data table. Note: Avoid confusing this with variable/object concept in computer programming.
Numerical (Quantitative) Variable A variables that contains numbers.
Categorical (Qualitative) Variable A variable that does not contain numbers (e.g. categories, text, etc).
Discrete Variable A numerical variable that can only contain values separated by specific intervals.
Continuous Variable A numerical variable where there are an infinite number of possible values between any two values.
Ordinal Variable A categorical variable that contains ordered values.
Nominal Variable A categorical variable that contains non-ordered values.
Time Series A type of continuous (numeric) variable based on time.

Bar (Column) Plot A visualization that displays the shape of data by representing the frequency of specific values as bars.
Measure of Center Statistics that describe the center or typical value in the data.
Mean (Average) The sum of the data divided by the number of values.
Median The middle of a ordered sequence of values.
Mode The value that appears most in the data.
Modality The number of “meaningful peaks” in your data.
Skewness A description of whether the data is symmetrical.
